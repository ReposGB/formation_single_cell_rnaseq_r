# Répertoire racine contenant tous les échantillons
input_dir <- "GSE155698_RAW"

# Liste des sous-répertoires (échantillons)
sample_dirs <- list.dirs(input_dir, recursive = FALSE)

# Créer une liste d’objets Seurat
seurat_list <- lapply(sample_dirs, function(dir) {
  
  sample_name <- basename(dir)  # nom de l’échantillon = nom du dossier

  print(paste("read sample ", sample_name))
  
  cur_data <- Read10X(data.dir = file.path(dir, 'filtered_feature_bc_matrix'))

  seu <- CreateSeuratObject(counts = cur_data, project = sample_name, min.cells = 3, min.features = 200)
  
  seu$orig.ident <- sample_name
  
  return(seu)
})

seurat.norm_3.pdac_15 <- NormalizeData(seurat.norm_3.pdac_15, normalization.method = "LogNormalize", scale.factor = 10000) 
seurat.norm_3.pdac_15 <- FindVariableFeatures(seurat.norm_3.pdac_15, selection.method = "vst", nfeatures = 2000)
seurat.norm_3.pdac_15 <- ScaleData(seurat.norm_3.pdac_15)

DimPlot(seurat.norm_3.pdac_15, reduction = "pca")

Detection automatisée des types cellulaires - Méthode “SingleR”

Nous allons utiliser le package SingleR.

Charger les librairies suivantes:

library(SingleCellExperiment)
library(SingleR)
library(celldex) # celldex fournit des jeux de référence annotés de transcriptomes de types cellulaires humains ou murins.

SingleR attend des données log-normalisées au format SinGleCellExperiment. On transforme notre objet Seurat en objet SinGleCellExperiment grace à la fonction as.SingleCellExperiment.

Vu qu’il y a des warnings, on vérifie bien que l’objet scecontient bien des données.
